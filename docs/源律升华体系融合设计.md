# 一、总定位

## 1.1 体系名称

本融合体系暂定名：

```text
源律升华体系
Source Law Sublimation
```

其中：

```text
源 = 源质，世界底层超凡物质
律 = 符文、仪式、术式、遗迹蓝图中可被解析和复用的规律
升华 = 玩家把源质规律内化进身体和器官网络的成长路径
```

## 1.2 一句话定义

**源律升华体系是一套以源质材料为底层、以器官承载为成长核心、以符文术式为外部表达、以以太工业为稳定化工程、以灵能解析为信息突破的统一力量系统。**

它不是两套并列魔法，而是一套力量的四种使用方式：

| 使用方式 | 代表系统 | 核心体验 |
|---|---|---|
| 内化 | 升华器官、魔药、能级、道路 | 长期成长、构筑、风险 |
| 随身 | 魔符、源律武装、法力、即时施法 | 战斗、采矿、移动、探索 |
| 定点 | 祭坛、仪式、机器祝福、传送 | 基地经营、战略效果 |
| 工业 | 以太机器、符文量产、以太武装 | 自动化、高效率、高阶装备 |

## 1.3 设计目标

1. 保留源质升华体系的成长深度：路线、器官、能级、魔药、畸变、遗迹、灵能。
2. 保留神学魔法体系的操作特色：符文、魔符、源律武装、祭坛仪式、双轨施法。
3. 让法力、符文、祭坛都成为源质规律的表达，不另起一套互相竞争的魔法底层。
4. 让工业线与升华线保持“弱依赖、强联动”：工业可独立推进，源律术式必须依附于升华器官或外源供能。
5. 支持当前项目架构：game-owned C++ 玩法逻辑、AngelScript 内容定义、客户端表现层、稳定值事件和显式生命周期边界。
6. 第一阶段优先做可玩的闭环，不一开始追求大而全。

# 二、融合原则

## 2.1 最重要原则

```text
源质有元素，器官有状态，术式有结构。
```

这条原则用于避免体系压扁。

不能把稳定、畸变、灵能直接当成元素。它们不是“土水火风光暗秩序混沌”中的一类，而是器官网络在承载源质时表现出的状态与能力。

更准确的关系是：

```text
源质元素：决定能量性质
器官网络：决定身体如何承载、转化、输出源质
器官状态：决定稳定、畸变、灵能、精神负荷等风险和上限
术式结构：决定外部施法、仪式、符文编译方式
```

## 2.2 升华为源，术式为用

融合后不再保留“纯术式玩家”作为独立通关路线。原因很简单：

```text
没有升华器官 → 没有稳定个人源质通道
没有个人源质通道 → 没有稳定法力
没有稳定法力 → 不能主动运行随身术式
```

术式不是独立力量来源，而是升华者或外源设备调用源律的方式。

```text
升华玩家：通过魔药、器官、魔兽素材提升源质储备和器官网络，并获得个人法力
术式分支：升华后用符文、魔符、源律武装、祭坛把源质能力外放和模块化
工业玩家：可通过 GT 机器、以太设备、自动化和武装推进普通工业与以太工业
融合玩家：用器官提供法力，用符文调律器官，用以太工业量产源律组件，获得效率、稳定性、产量、上限和特殊能力的超额收益
```

工业线与升华线可以弱依赖，但术式分支与升华不是弱依赖关系。个人术式必须至少完成启灵，或者借助外源供能装置运行。

三条路线的交汇点必须显式存在：

| 交汇点 | 作用 |
|---|---|
| 器官 → 法力 | 心脏、血液、神经等器官决定法力上限、回复、冷却和承载风险 |
| 符文 → 器官 | 符文可用于魔药校准、器官调律、降低冲突和污染 |
| 源质 → 符文 | 源质材料可封装成符文，成为外部术式材料 |
| 祭坛 → 器官 | 祭坛可进行调律、净化、共鸣，辅助升华但不能替代升华 |
| 以太 → 术式 | 以太机器可量产符文、魔符、魔药材料和术式装备 |
| 灵能 → 全系统 | 灵能读取源律信息，提升遗迹解析、仪式预览、兼容率判断 |

凡人阶段能做的事情应限制为：

```text
采集水晶、符石、源质材料
制作空白符石、低阶符文胚
操作外源供能的机器或祭坛辅助流程
准备启灵魔药和器官样本
不能主动用个人法力释放源律武装术式
```

## 2.3 不抢底层位置

各概念的层级必须明确：

| 概念 | 层级 | 说明 |
|---|---|---|
| 源质材料 | 世界底层物质（物品） | 生物、矿脉、生态、遗迹都围绕它变化；可开采、采集、存储、交易 |
| 源质储备 | 玩家身体内化源质（面板数值） | 通过魔药/器官吸收源质材料来提升；决定法力上限和器官成长 |
| 以太 | 工业稳定化源质（从源质材料转化） | 适合机器、武装、遗迹设备；不可从源质储备转化，不可被身体吸收 |
| 法力 | 升华者即时施法资源（HUD 条） | 升华器官网络即时调动源质术式的安全额度 |
| 灵能 | 器官相关信息能力 | 解析、预判、感知、蓝图研究 |
| 符文 | 源质规律封装 | 稳定、可存储、可组合 |
| 魔符 | 术式组件 | 源律武装编译用的消耗式组件 |
| 祭坛 | 大型源质回路 | 定点、慢速、强效果 |
| 升华器官 | 内化承载器 | 决定成长上限、风险、能力风格 |

## 2.4 术式武装不是升华的替代品

源律武装不能让普通玩家绕过升华直接施法。它只是术式载体，真正驱动它的是升华者的器官网络、法力，或外源供能装置。

```text
普通人 + 源律武装 = 只能装配、维护、预设，不能用个人法力主动施法
升华者 + 器官 = 身体内化源质规律
升华者 + 源律武装 = 用身体能力强化术式输出
工业玩家 + 以太机器 = 用机器稳定和放大源质规律
普通人 + 外源供能装置 = 可触发有限外源术式，但不获得个人法力
```

正式命名建议：

```text
源律武装：随身术式载体总称，可手工制作低阶版本，也可工业化升级；主动运行需要升华法力或外源供能。
以太武装：以以太核心驱动的高阶源律武装，是工业化后的战斗/工具形态。
魔法书：旧方案名或早期原型名，不作为最终体系正式命名。
```

# 三、三层底层模型

## 3.1 源质元素层

源质元素层回答：

```text
这股力量是什么性质？
```

它包括火、水、土、风、光、暗、秩序、混沌等元素，用于符文、魔符、仪式、源质素材、魔药属性、道路倾向。

## 3.2 器官网络层

器官网络层回答：

```text
玩家用什么身体结构承载它？
```

它包括心脏、骨骼、血液、肺、眼睛、神经、皮肤等升华器官。

同一种元素，被不同器官承载会产生不同结果：

| 元素 | 器官 | 可能结果 |
|---|---|---|
| 火 | 心脏 | 源质爆发、过热、火焰抗性 |
| 火 | 血液 | 灼烧攻击、体温风险、燃血 |
| 水 | 肺 | 水下呼吸、环境适应 |
| 水 | 血液 | 恢复、流动、污染稀释 |
| 土 | 骨骼 | 护甲、承压、稳定 |
| 风 | 神经 | 反应、闪避、技能冷却 |
| 光 | 眼睛 | 源质视野、灵能感知 |
| 暗 | 皮肤 | 潜行、侵蚀抗性、污染遮蔽 |

## 3.3 器官状态层

器官状态层回答：

```text
身体承载得怎么样？
```

它包括：

```text
稳定度
畸变度
灵能适配
精神负荷
器官兼容率
源质纯净度
污染耐受
```

这些不是元素，而是器官网络的状态。

## 3.4 术式结构层

术式结构层回答：

```text
外部如何调用这条规律？
```

它包括：

```text
符文：封装源质规律
魔符：把符文转成施法组件
源律武装：编译魔符并即时释放术式
祭坛：把符文排列成大型回路
蓝图：遗迹文明保存的高阶术式工程
以太机器：把术式稳定化、自动化、工业化
```

# 四、核心资源与力量形态

## 4.1 五个核心量

| 名称 | 定位 | 主要用途 |
|---|---|---|
| 源质材料 | 世界中的超凡物质（物品） | 可开采、采集、存储、交易；用于魔药、器官、符文封装、工业转化 |
| 源质储备 | 玩家身体已内化的源质总量（面板数值） | 决定法力上限、器官成长、能级突破；通过魔药/器官吸收源质材料来提升 |
| 以太 | 工业稳定化源质（从源质材料转化） | 机器、武装、遗迹设备、自动化；不可从源质储备转化，不可被身体吸收 |
| 法力 | 升华者即时施法资源（HUD 条） | 源律武装施法、祭坛启动、器官技能外放；由源质储备通过器官网络转化而来 |
| 灵能 | 器官相关信息能力 | 预警、探测、解析、蓝图、研究 |
| 源律 | 可复用的源质规律 | 符文、魔符、仪式、遗迹技术 |

## 4.2 相互关系

```text
源质材料 → 魔药 / 器官吸收 → 源质储备（面板数值）
源质储备 → 器官网络转化 → 法力（HUD 条）
源质材料 → 符文封装 → 魔符 / 祭坛
源质材料 → 工业稳定化 → 以太
以太 → 机器 / 武装 / 高阶遗迹装置
法力 → 玩家即时术式输出额度
灵能 → 对源律结构的解析、预判和读取
```

## 4.3 转化建议

| 转化 | 可行性 | 风险 |
|---|---|---|
| 源质材料 → 以太 | 可工业化，效率中等 | 需要机器和稳定剂 |
| 以太 → 源质材料 | 可行但高风险 | 可能污染器官、增加畸变 |
| 源质材料 → 符文 | 稳定封装 | 需要符石、水晶、雕刻或蚀刻 |
| 符文 → 魔符 | 术式转译 | 魔符插入源律武装后消耗 |
| 源质材料 → 源质储备 | 通过魔药/器官吸收 | 受器官兼容率和纯净度影响 |
| 源质储备 → 法力 | 器官网络即时转化 | 受法力上限和冷却限制 |
| 灵能 → 蓝图信息 | 信息解析 | 增加精神负荷 |

## 4.4 玩家主循环

```text
探索世界
  ↓
发现源质生态 / 水晶矿 / 魔兽巢穴 / 遗迹
  ↓
采集源质材料、符石、水晶、魔兽器官
  ↓
选择路线：
  ├─ 内化：源质材料萃取 → 魔药 → 器官吸收 → 源质储备提升
  ├─ 术式：水晶 → 符文 → 魔符 → 源律武装
  ├─ 仪式：符文 → 祭坛排列 → 定点效果
  └─ 工业：源质材料 → 机器处理 → 以太 → 自动化 / 武装
  ↓
提升战斗、采矿、探索、基地能力
  ↓
挑战更强魔兽、污染区、遗迹和 Boss
```

# 五、源质元素体系

## 5.1 基础八元素

融合后建议沿用神学魔法系统中的 8 元素，作为源质、符文、魔符、祭坛和道路倾向的统一元素表。

| 元素 | 核心性质 | 术式方向 | 升华方向 | 工业联动方向 |
|---|---|---|---|---|
| 火 | 加速、破坏、热量 | 伤害、熔炼、爆发 | 炎炉、燃血、过热 | 提升机器速度 |
| 水 | 流动、恢复、冷却 | 治疗、净化水流、冷却 | 潮汐、恢复、适应 | 降低机器损耗 |
| 土 | 稳固、承压、结构 | 采掘、防御、地形 | 沙铠、骨骼、防御 | 提升产出倍率 |
| 风 | 移动、范围、效率 | 位移、浮空、冲刺 | 风暴、神经、反应 | 降低能耗 |
| 光 | 揭示、纯净、祝福 | 照明、驱散、治疗 | 辉光眼、净化器官 | 机器免维护、稳定显示 |
| 暗 | 隐匿、衰败、侵蚀 | 诅咒、吸取、潜行 | 暗影器官、污染耐受 | 特殊诅咒效果 |
| 秩序 | 结构、精确、控制 | 屏障、锁定、校准 | 器官协调倾向 | 提升配方成功率 |
| 混沌 | 随机、转化、力量 | 随机倍率、突变 | 高风险突变倾向 | 随机翻倍、过载 |

## 5.2 雷与生命的处理

源质升华文档中有雷、生命等属性。融合后建议不强行扩张成十多个基础元素，而是作为复合谱系或生物谱系。

| 原属性 | 融合后定位 |
|---|---|
| 雷 | 风 + 火的高能流动，或风 + 秩序的定向放电 |
| 生命 | 水 + 光的生物谱系，也可作为器官培育专用源质纯度 |
| 稳定 | 器官状态，不是元素 |
| 畸变 | 器官状态，不是元素 |
| 灵能 | 器官信息能力，不是元素 |

## 5.3 元素相性：相生、相克与强冲突

元素之间需要有相生相克，用来计算不同器官之间的兼容率。它不改变“稳定、畸变不是元素”的原则，而是表示：

```text
元素相性 = 器官网络稳定度 / 畸变度的输入因子之一
最终结果 = 器官状态变化
```

### 相性等级

| 相性 | 条件 | 稳定度 | 畸变 |
|---|---|---|---|
| 同源 | 两个器官主元素相同 | 高 | 低 |
| 相生 | 两个器官元素能形成循环或互补 | 较高 | 较低 |
| 中性 | 无明显促进或冲突 | 不变或轻微波动 | 不变 |
| 相克 | 元素性质互相压制 | 降低 | 上升 |
| 强冲突 | 全身只有两种完全相反元素，且缺少主导/中和元素 | 大幅降低 | 大幅上升 |

### 推荐相生关系

相生关系用于鼓励多元素构筑，但不让玩家无脑堆同元素。

| 元素 | 相生元素 | 解释 |
|---|---|---|
| 火 | 风、秩序 | 风助燃，秩序约束火的输出 |
| 风 | 火、水 | 风助火势，也推动水的流动 |
| 水 | 木性生命谱系、光、风 | 水支持恢复，光净化，风带来流动 |
| 土 | 水、秩序 | 水润土，秩序强化结构 |
| 光 | 水、秩序 | 光净化水，秩序让光稳定聚焦 |
| 暗 | 混沌、土 | 暗适合隐匿和沉积，混沌提供变化 |
| 秩序 | 土、光、火 | 秩序稳定结构、光学解析和能量输出 |
| 混沌 | 暗、火、风 | 混沌放大侵蚀、爆发和变化 |

说明：

```text
生命不作为基础元素时，可视为水 + 光的生物谱系。
雷不作为基础元素时，可视为风 + 火或风 + 秩序的复合谱系。
```

### 推荐相克关系

| 元素 | 相克元素 | 解释 |
|---|---|---|
| 火 | 水、暗 | 水压制火，暗侵蚀火的可控性 |
| 水 | 火、混沌 | 火蒸发水，混沌污染水的恢复性 |
| 土 | 风、混沌 | 风蚀土，混沌破坏结构 |
| 风 | 土、秩序 | 土阻风，秩序限制自由流动 |
| 光 | 暗、混沌 | 光揭示和净化暗/混沌 |
| 暗 | 光、秩序 | 光驱散暗，秩序限制侵蚀 |
| 秩序 | 混沌、风 | 秩序与混沌根本冲突，也约束风 |
| 混沌 | 秩序、光、水 | 混沌破坏规则、净化和恢复 |

### 强冲突规则

强冲突不是简单的“出现相克元素就爆炸”，而是指器官网络中只有两种完全相反元素，并且缺少主导元素或中和元素。

典型强冲突：

| 组合 | 风险 |
|---|---|
| 火 + 水 | 过热/骤冷反复，源质流断裂 |
| 光 + 暗 | 净化与侵蚀互相撕扯 |
| 秩序 + 混沌 | 结构与突变直接对冲 |
| 土 + 风 | 承压结构与高速流动互相破坏 |

强冲突判定建议：

```text
如果器官网络中只有两种主元素
且这两种元素属于强冲突对
且两者权重接近
且没有第三元素作为中和或主导
则稳定度大幅下降，畸变增长倍率提高。
```

如果加入第三种中和元素，强冲突可降为普通相克：

| 冲突 | 中和方式 |
|---|---|
| 火 + 水 | 加入秩序做节流，或加入土做承压 |
| 光 + 暗 | 加入秩序做边界，或加入水做净化缓冲 |
| 秩序 + 混沌 | 加入土做结构承载，或加入光做净化监控 |
| 土 + 风 | 加入水做流体缓冲，或加入秩序做导流结构 |

### 数值建议

可以先用简单权重：

| 相性 | 稳定修正 | 畸变修正 |
|---|---:|---:|
| 同源 | +12 | -8 |
| 相生 | +8 | -5 |
| 中性 | 0 | 0 |
| 相克 | -10 | +8 |
| 强冲突 | -25 | +20 |

最终公式示例：

```text
器官兼容率 =
基础兼容率
+ 同路线加成
+ 器官品质加成
+ 元素相性修正
+ 符文调律加成
- 污染惩罚
- 槽位冲突惩罚
```

## 5.4 源质纯度

| 纯度 | 用途 | 风险 |
|---|---|---|
| 污浊源质 | 低阶合成、应急能源 | 畸变风险高 |
| 粗源质 | 普通材料处理 | 可离心分离 |
| 属性源质 | 魔药、符文、器官培养 | 需要元素匹配 |
| 纯净源质 | 高级魔药、稳定器官 | 成本高 |
| 活化源质 | 器官培育、生命谱系 | 需要控制污染 |
| 古代源质 | 遗迹、高能级突破 | 需要灵能解析 |
| 以太化源质 | 工业侧高阶能源 | 可进入以太机器 |

# 六、器官网络与升华者

## 6.1 升华者定位

升华者不是“会施法的人”，而是把源质规律内化进身体的人。

```text
外部术式 = 借工具调用源律
升华器官 = 身体本身成为源律回路
```

|---|---|---|
| 0 | 凡人 | 可采集、制作符文胚、操作机器和外源祭坛，但无个人法力和器官能力 |

| 2 | 第二能级：凝辉 | 器官网络成熟，路线特性明显 |

## 6.6 混搭与兼容

同路线器官兼容度高，不同路线混搭有收益也有风险。混搭判断不只看路线，也要看器官主元素之间的相性：

```text
同路线 + 同元素：最稳定，畸变最低
不同路线 + 相生元素：稳定较高，可形成复合玩法
不同路线 + 相克元素：稳定下降，畸变上升
全身只有两种强冲突元素：稳定大幅下降，畸变快速积累
```

示例：

| 组合 | 结果 |
|---|---|
| 沙铠心脏 + 沙铠骨骼 | 稳定度提升，护盾增强 |
| 沙铠心脏 + 风暴神经 | 冲撞和闪避共鸣，但稳定度下降 |
| 炎炉心脏 + 潮汐肺 | 高冲突，过热与冷却对抗，畸变风险上升 |
| 辉光眼 + 任意道路 | 灵能解析和污染识别增强 |
| 净化骨 + 炎炉血液 | 降低燃血副作用，但爆发上限略降 |

更精确的示例：

| 器官组合 | 元素关系 | 结果 |
|---|---|---|
| 沙铠岩核 + 净岩晶骨 | 同源土 | 稳定高，畸变低，防御路线纯化 |
| 沙铠岩核 + 辉光眼 | 土 + 光，中性偏相生 | 探测能力增强，稳定小幅提升 |
| 炎炉心脏 + 风暴神经 | 火 + 风，相生 | 爆发和机动增强，稳定可控 |
| 炎炉心脏 + 潮汐肺 | 火 + 水，强冲突候选 | 若无秩序/土中和，稳定低、畸变高 |
| 辉光眼 + 暗影皮肤 | 光 + 暗，强冲突候选 | 可做高风险侦察/潜行构筑，但需要秩序边界 |
| 秩序骨骼 + 混沌血液 | 秩序 + 混沌，强冲突候选 | 高风险突变构筑，必须依赖调律 |

# 七、稳定、畸变与灵能

## 7.1 三者定位

| 名称 | 本质 | 关联 |
|---|---|---|
| 稳定度 | 器官网络承载源质的安全程度 | 器官品质、槽位兼容、魔药纯度 |
| 畸变度 | 器官网络被污染、冲突、错误源律扭曲的程度 | 污染源质、冲突器官、失败仪式 |
| 灵能 | 器官网络对信息侧源律的读取能力 | 眼睛、神经、心脏、精神负荷 |

它们都与器官相关，不是元素。

## 7.2 稳定度

稳定度影响：

```text
技能失败概率
源律武装施法反噬概率
魔药成功率
器官成长效率
畸变增长速度
以太武装过载反噬
祭坛仪式个人承载能力
```

稳定度来源：

```text
高品质器官
同路线器官
同元素器官
相生元素器官
纯净源质
稳定剂
净化类器官
秩序/光相关术式辅助
祭坛调律仪式
```

注意：秩序和光可以帮助稳定，但稳定不是秩序或光元素本身。

稳定度下降来源：

```text
相克元素器官
强冲突元素组合
只有两种完全相反元素且权重接近
冲突路线器官
污染源质
失败调律
```

## 7.3 畸变度

畸变来源：

```text
服用低品质魔药
植入冲突器官
植入相克元素器官
形成强冲突元素网络
使用污染源质
长时间待在污染区
仪式失败
以太反转源质
强行突破能级
混沌术式过量使用
```

畸变阶段：

| 畸变值 | 状态 | 效果 |
|---|---|---|
| 0-20 | 正常 | 无明显副作用 |
| 21-40 | 轻微污染 | 偶发视野扭曲、源质恢复波动 |
| 41-60 | 畸变症状 | 器官副作用、法术偏移 |
| 61-80 | 严重畸变 | 技能失控、精神负荷上升 |
| 81-100 | 源质失控 | 可能死亡、暴走或生成畸变体 |

## 7.4 灵能

灵能是器官对源律信息的读取能力，主要服务于：

```text
危险提示
源质视野
矿脉探测
敌人预判
遗迹解析
蓝图研究
祭坛配方提示
机器异常诊断
```

灵能不等于法力，不等于光元素，也不等于秩序元素。

更准确地说：

```text
辉光眼可以增强灵能感知
雷鸣神经可以增强战斗预判
稳定心脏可以降低解析反噬
精神负荷决定灵能使用上限
```

## 7.5 精神负荷

灵能和复杂术式都会增加精神负荷。

负荷来源：

```text
遗迹解析
高阶源律武装施法
长时间维持射线/领域
失败仪式
混沌符文过量
使用超出能级的蓝图
```

负荷过高后果：

```text
错误提示
画面噪声
祭坛配方误判
源质视野产生假影
技能冷却延长
吸引畸变体或遗迹守卫
```

# 八、法力系统

## 8.1 法力定位

法力是升华者个人的源质术式缓冲池。

它不是世界底层资源，也不接管道网络，不作为可储存在方块中的通用能源。

```text
法力 = 升华器官网络可安全即时调动的源质流量
```

更精确地说：

```text
源质决定“力量从哪里来”
器官决定“身体能承载多少、转换多快、风险多大”
法力决定“当前这一刻器官网络能安全输出多少术式”
```

所以不存在一条和源质升华并列的“自然法力成长线”。玩家没有启灵和器官网络时，个人法力为 0。法力只有恢复、扩容、调律和临时增益；这些变化最终都要落回器官、祭坛、装备、符文或以太工具。

## 8.2 法力与源质的区别

| 维度 | 源质材料 | 法力 |
|---|---|---|
| 层级 | 世界底层超凡物质（物品） | 升华者个人输出缓冲（HUD 条） |
| 是否可采集 | 可以，来自矿脉、生态、魔兽、遗迹 | 不作为独立矿物或世界资源采集 |
| 是否能存方块 | 可以存入材料、容器、机器 | 不接管道，不做法力电池网络 |
| 主要用途 | 魔药、器官、符文、以太、生态、升华 | 源律武装施法、祭坛启动、器官技能外放 |
| 成长方式 | 提升源质储备、纯度、器官承载、道路能级 | 启灵后由器官、祭坛、装备、仪式改变上限和回复 |
| 风险 | 污染、畸变、器官冲突 | 法力透支、精神负荷、术式失败 |

换句话说：

```text
源质材料是燃料和材料（物品，可采集可存储）。
源质储备是身体已内化的源质总量（面板数值，决定法力上限和器官成长）。
以太是从源质材料工业稳定化后的形态（机器能源，不可被身体吸收）。
法力是升华器官网络的安全输出阀（HUD 条，即时施法消耗）。
```

## 8.3 法力与器官

凡人没有个人法力。启灵后，玩家的第一个核心器官才会形成稳定源质通道，并产生个人法力池。

升华器官可以影响法力：

| 器官 | 可能影响 |
|---|---|
| 心脏 | 法力上限、爆发输出 |
| 神经 | 施法冷却、蓄力速度 |
| 眼睛 | 目标锁定、源质视野法术成本 |
| 血液 | 法力回复、药水效果 |
| 骨骼 | 高阶仪式承载能力 |

因此法力的多数“成长”都应该解释为器官网络变强后，安全输出阀变宽了。

## 8.4 法力与符文 / 祭坛交汇

符文和祭坛不生产独立的法力体系，而是改变玩家调用源律的效率。

| 交汇 | 效果 |
|---|---|
| 秩序增幅魔符 | 降低同一法术所需法力 |
| 水增幅魔符 | 降低冷却或延长持续时间，相当于提高法力效率 |
| 祭坛共鸣 | 启灵后在祭坛核心附近加快法力回复 |
| 法力扩容仪式 | 永久拓宽器官网络输出阀，但应受稳定度或器官承载限制 |
| 器官调律仪式 | 降低施法反噬和精神负荷 |
| 灵能预览 | 提前显示仪式法力消耗与风险 |

## 8.5 法力来源

| 来源 | 描述 | 定位 |
|---|---|---|
| 器官基础回复 | 启灵后核心器官的低速恢复 | 保证升华者能使用低阶术式 |
| 法力药水 | 瞬间回复 | 消耗品 |
| 法力水晶 | 一次性完全回复 | 探索奖励 |
| 祭坛共鸣 | 站在祭坛旁回复加快 | 基地收益 |
| 献祭仪式 | 消耗生命或物品换法力 | 风险收益 |
| 器官自循环 | 高能级器官少量回复 | 升华收益 |

其中“器官基础回复”来自启灵器官，不应解释成玩家凭空修炼出一套独立魔法等级。

## 8.6 法力上限扩容

| 方式 | 增加量 | 说明 |
|---|---|---|
| 祭坛扩容仪式 | +25 | 启灵后可获得，必须检查稳定度 |
| 法力容器 | +50 | 工业联动制造，本质是外置输出调节器 |
| 心脏器官成长 | 按品质提升 | 升华路线收益 |
| 遗迹蓝图改造 | 高阶扩容 | 中后期内容 |

扩容规则建议：

```text
最终法力上限 =
启灵器官基础法力上限
+ 心脏 / 血液 / 神经等器官加成
+ 祭坛扩容仪式加成
+ 法力容器或装备加成
+ 遗迹蓝图改造加成
- 畸变和精神负荷惩罚
```

也就是说，术式分支可以通过祭坛和装备提高法力效率，但法力本身来自升华器官；器官、符文、祭坛和以太武装交汇时才最强。

# 九、符文系统

## 9.1 符文定位

符文是被稳定封装的源质规律片段。

```text
源质材料 → 提纯 / 雕刻 / 蚀刻 → 符文
```

符文既可用于祭坛，也可转化为魔符，或进入工业机器作为源律组件。

## 9.2 符文等级

沿用原神学魔法设计：

| 等级 | 强度值 | 后缀 |
|---|---|---|
| 普通 | 1 | `_common` |
| 精良 | 2 | `_refined` |
| 卓越 | 3 | `_superior` |
| 传奇 | 5 | `_legendary` |

共：

```text
8 元素 × 4 等级 = 32 种符文
```

## 9.3 双轨生产路线

### 手动路线

适合前期玩家准备启灵材料，也适合升华者在不依赖工业时制作术式组件：

```text
神秘水晶矿 / 源质晶体
  ↓
粗水晶 / 空白符石
  ↓
手工雕刻台
  ↓
普通符文
  ↓
手工融合台
  ↓
高级符文
```

特点：

```text
不依赖工业
速度慢
产出稳定但无加成
适合作为启灵前准备和启灵后术式分支入口
```

### 工业路线

适合 GT / 自动化玩家：

```text
神秘水晶矿 / 源质矿脉
  ↓
采矿机
  ↓
破碎机
  ↓
离心机分离元素精华
  ↓
符文蚀刻机
  ↓
符文融合器
```

特点：

```text
产量更高
速度更快
可接入 AE2 自动合成
需要电力、机器和配方
```

## 9.4 与源质材料的关系

| 源质材料 | 符文用途 |
|---|---|
| 土源质晶体 | 土符文 |
| 水源质精华 | 水符文 |
| 火源质精华 | 火符文 |
| 风暴晶体 | 风符文或复合雷系术式材料 |
| 净化花粉 | 光符文辅助材料 |
| 畸变肉块 | 暗/混沌符文高风险材料 |
| 古代源质 | 传奇符文、遗迹蓝图激活 |

# 十、魔符与源律武装

## 10.1 魔符定位

魔符是符文转译后的施法组件。

```text
符文 = 源律片段
魔符 = 可被源律武装编译的术式组件
```

魔符分三类：

```text
形态魔符：决定如何送达
效果魔符：决定产生什么效果
增幅魔符：修饰范围、伤害、冷却、消耗等参数
```

## 10.2 形态魔符

| 形态 | 描述 |
|---|---|
| 弹射 | 向前发射弹丸 |
| 自施 | 对自身立即生效 |
| 范围 | 对目标区域生效 |
| 射线 | 持续光束 |
| 触碰 | 近身接触目标 |

## 10.3 效果魔符

| 元素 | 效果 |
|---|---|
| 火 | 伤害、燃烧、熔炼 |
| 水 | 恢复、冷却、净化液流 |
| 土 | 采掘、护盾、地形 |
| 风 | 位移、浮空、加速 |
| 光 | 照明、驱散、揭示 |
| 暗 | 诅咒、潜行、生命汲取 |
| 秩序 | 屏障、锁定、校准 |
| 混沌 | 随机倍率、突变、转化 |

## 10.4 增幅魔符

| 元素 | 增幅名称 | 效果 |
|---|---|---|
| 火 | 烈怒 | 增加伤害或采掘强度 |
| 水 | 延流 | 增加持续时间或降低冷却 |
| 土 | 广域 | 增加范围 |
| 风 | 疾速 | 降低施法时间 |
| 光 | 精准 | 追踪或提高命中 |
| 暗 | 穿透 | 无视部分防御 |
| 秩序 | 节能 | 降低法力消耗 |
| 混沌 | 变数 | 随机倍率 |

## 10.5 源律武装等级

“魔法书”不再作为正式装备命名。随身施法载体统一归入“源律武装”，并允许从低阶手工导具成长到高阶以太武装。

| 等级 | 槽位 | 预设数 | 法力倍率 |
|---|---|---|---|
| 基础源律导具 | 3：形态 1 + 效果 1 + 增幅 1 | 2 | 1.0 |
| 源律臂铠 | 4：形态 1 + 效果 1 + 增幅 2 | 3 | 0.85 |
| 以太术式武装 | 5：形态 1 + 效果 1 + 增幅 3 | 4 | 0.7 |

命名方向：

| 载体 | 定位 |
|---|---|
| 基础源律导具 | 早期手工制作，可理解为符文导杖、符文护符、术式手套等 |
| 源律臂铠 | 中期战斗/工具载体，开始与装备槽和器官共鸣绑定 |
| 以太术式武装 | 高阶工业版本，使用以太核心、模块槽和过载机制 |

## 10.6 消耗式插槽规则

保留原设计特色：

```text
插入魔符 → 魔符即时消耗
覆盖魔符 → 新魔符消耗，旧配置销毁
切换预设 → 不额外消耗
```

这让源律武装成为长期构筑选择，而不是无成本随意换技能。

## 10.7 与升华器官的共鸣

升华器官可以改变源律武装表现，但不取代源律武装。

示例：

| 器官 | 对源律武装的影响 |
|---|---|
| 雷鸣神经 | 降低风系和雷系术式冷却 |
| 沙铠岩核 | 土系护盾魔符强度提升 |
| 潮汐肺 | 水系恢复魔符法力消耗降低 |
| 辉光眼 | 光系揭示魔符范围提升 |
| 炽热心脏 | 火系魔符伤害提升，但过热风险上升 |

## 10.8 法术示例

| 配置 | 效果 |
|---|---|
| 弹射 + 火 + 火 + 秩序 + 风 | 追踪火球，较低法力消耗 |
| 范围 + 土 + 土 | 范围采矿 |
| 自施 + 水 + 水 + 水 | 强效治疗 |
| 自施 + 风 + 风 + 水 | 高速移动，持续时间增加 |
| 弹射 + 暗 + 暗 + 光 | 穿透诅咒并标记目标 |
| 射线 + 混沌 + 混沌 + 火 | 随机倍率高伤害射线 |

# 十一、祭坛仪式

## 11.1 祭坛定位

祭坛是大型定点源质回路。

它不是随身战斗技能，而是用于：

```text
机器祝福
工具附魔
基地强化
地形改造
传送链接
占卜
遗迹门开关
法力扩容
器官调律
污染净化
```

## 11.2 基础祭坛与高级祭坛

保留原神学魔法系统结构：

```text
基础祭坛：核心 + 4 台座
高级祭坛：核心 + 8 台座
```

高级祭坛可以由工业制造的高级台座升级，也可以通过高阶仪式路线制作更昂贵版本。

## 11.3 仪式执行流程

```text
玩家放置符文到台座
  ↓
系统实时匹配仪式配方
  ↓
灵能足够时显示更多预览信息
  ↓
玩家站在核心祭坛确认
  ↓
检查法力、稳定度、污染、符文等级
  ↓
播放 3-10 秒仪式动画
  ↓
生效，按配方决定是否消耗符文
```

## 11.4 仪式配方示例

| 符文排列 | 效果 | 类型 |
|---|---|---|
| 火·秩序·火·秩序 | 机器加速 | 机器祝福 |
| 水·水·水·水 | 机器冷却 / 免维护 | 机器祝福 |
| 土·秩序·光·秩序 | 机器增产 | 机器祝福 |
| 光·土·光·土 | 占卜矿脉 | 灵能占卜 |
| 光·光·暗·暗 | 遗迹门开关 | 世界事件 |
| 风·风·风·风 | 玩家移速提升 | 玩家增益 |
| 火·土·水·风 | 祭坛传送链接 | 传送 |
| 秩序·秩序·秩序·秩序 | 法力上限提升 | 永久成长 |
| 光·水·秩序·土 | 器官调律，降低畸变 | 升华辅助 |
| 混沌·混沌·秩序·光 | 高风险源质重组 | 转化 |

## 11.5 器官与祭坛

祭坛可以对器官做三类事情：

```text
调律：提高稳定度，降低冲突
净化：降低污染和畸变
共鸣：临时增强某一器官或道路
```

但核心升华突破仍需要魔药、器官素材和玩家参与，不能被祭坛完全自动化。

# 十二、魔药与器官培育

## 12.1 魔药定位

魔药是内化源质规律的关键道具。

| 类型 | 作用 |
|---|---|
| 启灵魔药 | 从凡人进入第一能级 |
| 强化魔药 | 提升器官成长或品质 |
| 晋升魔药 | 突破到下一能级 |
| 调律药剂 | 降低冲突，提高兼容 |
| 净化药剂 | 降低污染和畸变 |

## 12.2 制作流程

```text
魔兽素材 / 植物 / 矿物
  ↓
源质萃取
  ↓
属性分离
  ↓
加入稳定剂 / 符文校准
  ↓
魔药反应釜
  ↓
灵能检测兼容率
  ↓
服用 / 注射 / 器官灌注
```

## 12.3 符文参与魔药

为了融合两份文档，魔药可引入符文作为校准器。

示例：

| 魔药 | 原材料 | 符文校准 |
|---|---|---|
| 沙铠启灵魔药 | 岩蜥鳞片、土源质、晶化骨粉 | 土符文、秩序符文 |
| 潮汐启灵魔药 | 海蛇鳞、潮汐腺体、水源质 | 水符文、光符文 |
| 风暴启灵魔药 | 雷鸟羽毛、磁晶碎片、风/火源质 | 风符文、火符文 |
| 炎炉启灵魔药 | 炽核、火源质、熔血材料 | 火符文、混沌符文 |
| 辉光启灵魔药 | 辉光鹿角、净化花粉、光源质 | 光符文、秩序符文 |

符文不是必需全部参与低阶魔药，但在高阶魔药中应明显提高成功率和稳定性。

## 12.4 器官培育

器官培育可保留高天体系的“器官成长”特色，并接入工业机器：

```text
器官样本

  ↓
营养液
  ↓
符文调律
  ↓
器官培育舱
  ↓
升华器官
```

影响品质的因素：

```text
样本品质
源质纯度
营养液等级
符文等级
机器等级
玩家灵能辅助
污染控制
```

# 十三、以太科技与工业联动

## 13.1 以太定位

以太是从源质材料（非源质储备）通过工业手段稳定化后的形态，可被机器安全使用。

```text
RF / EU：普通工业能源
源质材料：世界中的超凡物质（物品，可开采采集）
源质储备：玩家身体已内化的源质（面板数值，不可工业化提取）
法力：玩家个人施法缓冲（HUD 条）
以太：从源质材料工业稳定化后的高阶能源（机器可用）
```

关键约束：

```text
源质材料 → 以太：可以，通过源质转化炉
源质储备 → 以太：不可以，身体内化的源质无法被机器提取
以太 → 源质储备：不可以，以太是工业形态，不能直接吸收进身体
```

以太不要直接替代 EU，也不要让源质直接等价为电。

# 十四、以太武装与高阶源律装备

## 14.1 装备分类

以太武装不是魔法书之外的另一条装备线，而是源律武装的工业化高阶形态。

```text
基础源律导具 → 源律臂铠 → 以太术式武装 / 以太武装
```

玩家启灵后可以用手工源律导具释放魔符术式；进入工业和遗迹阶段后，随身术式载体逐步升级为以太武装。这样既保留非工业制作入口，又让中后期装备统一到“以太武装”框架里。

设计结论：

```text
启灵早期：源律导具承担原“魔法书”的玩法入口。
中期：源律臂铠把魔符槽位装备化。
后期：以太武装全面替代独立施法书，成为战斗、工具、施法的一体化载体。
```

| 装备 | 定位 |
|---|---|
| 基础源律导具 | 早期随身术式载体 |
| 源律臂铠 | 中期多槽位术式载体 |
| 以太长剑 | 近战能量武器 |
| 以太火炮 | 远程爆发武器 |
| 源质护盾发生器 | 防御装备 |
| 灵能目镜 | 探测、解析、源质视野 |
| 机动背包 | 冲刺、悬浮、短飞行 |
| 轻型以太铠装 | 移动和闪避 |
| 重型以太铠装 | 护甲和抗性 |
| 术式核心装甲 | 源律武装/符文共鸣 |

## 14.2 模块槽

以太武装保留源律武装的魔符槽位，并额外加入工业模块：

```text
核心槽：决定装备等级
源质槽：决定元素属性
符文槽：决定术式效果
魔符槽：决定主动释放的形态、效果、增幅
稳定槽：降低过载风险
输出槽：增加伤害或效率
辅助槽：增加特殊功能
```

示例：

| 组合 | 效果 |
|---|---|
| 以太长剑 + 火源质核心 + 火符文 | 炽焰斩 |
| 以太长剑 + 风/火复合核心 + 秩序符文 | 雷鸣斩 |
| 源质护盾 + 土核心 + 秩序符文 | 稳定护盾 |
| 灵能目镜 + 光核心 + 辉光眼器官共鸣 | 遗迹扫描增强 |

## 14.3 过载机制

装备不能无限使用。

```text
连续使用 → 以太温度上升
温度过高 → 装备短暂失效
稳定度低 → 反噬玩家
源质属性冲突 → 畸变增加
魔符/符文过载 → 模块损坏或冷却延长
```

器官可以降低或放大过载：

```text
炎炉心脏：提高爆发上限，但热量风险上升
净化骨：降低污染反噬
```

计算建议：

```text

3. 同源 / 相生增加稳定并降低畸变。
4. 相克降低稳定并提高畸变。
5. 如果只有两种强冲突元素且权重接近，标记 has_severe_conflict。
6. 祭坛调律、秩序符文、净化器官可以降低强冲突惩罚，但不完全抹除。
```

计算公式：

```text

```

# 二十三、平衡与风险

## 23.1 三重技能限制

技能和法术建议有三重限制：

```text
资源消耗：法力 / 源质 / 以太
时间限制：冷却 / 蓄力 / 仪式时间
风险限制：稳定度 / 畸变 / 精神负荷 / 过载
```

示例：

```text
火球术：消耗法力，有冷却
岩盾：消耗源质，有冷却，受器官稳定影响
机器加速仪式：消耗法力和符文，持续 1 小时
以太火炮：消耗以太，连续使用会过载
遗迹解析：消耗精神负荷，失败可能吸引守卫
```

## 23.2 不让第一能级碾压

推荐强度：

```text
凡人：只能准备符文、材料和外源设备，不能主动运行个人术式
启灵者 + 基础源律导具：略强于普通工具，但受法力和器官稳定限制
第一能级：略强于铁/早期钢阶段
第二能级：强于钻石/中期装备，但仍怕 Boss
以太武装：强，但需要能源和维护
第三能级以上：中后期或终局内容
```

## 23.3 复杂度风险

风险：

```text
系统太多，玩家看不懂
符文、源质、法力、以太概念混乱
器官状态和元素被混淆
自动化吞掉手动仪式体验
第一版做太大
```

解决：

```text
统一术语：源质有元素，器官有状态，术式有结构
UI 显示下一步目标
JEI / 指南书展示流程
第一版只做少量道路和法术
高阶内容必须玩家参与
不确定问题优先加低频日志观察
```

# 二十四、最终推荐基座

本项目后续可以把以下句子作为力量体系总纲：

```text
源质材料是世界的超凡物质（物品，可开采采集存储）。
源质储备是身体已内化的源质总量（面板数值，决定法力上限和器官成长）。
法力是升华器官网络的安全输出阀（HUD 条，即时施法消耗）。
元素是源质的性质。
器官是玩家承载源质的身体网络。
稳定、畸变、灵能是器官网络的状态与能力。
符文是源质规律的稳定封装。
魔符和源律武装是随身术式编译器。
祭坛是定点大型源质回路。
以太是从源质材料工业稳定化后的形态（机器能源，不可被身体吸收）。
遗迹是旧文明保存的高阶源律工程。
```

对应玩法基座：

```text
探索生态 → 获取源质材料/水晶/器官素材
  ↓
手动或工业处理
  ↓
魔药/器官吸收 → 提升源质储备（面板数值）
  ↓
源质储备 → 器官网络转化 → 法力（HUD 条）
  ↓
启灵后用符文/魔符/源律武装提供即时能力
  ↓
祭坛提供基地和战略效果
  ↓
以太科技提供自动化和装备扩展
  ↓
灵能解析遗迹，解锁高阶源律
```

最推荐开发策略：

```text
先做第一条启灵道路，接入器官、稳定、畸变和个人法力；
再做符文 + 源律武装，让启灵者获得即时术式玩法；
再做源质机器和魔药链，形成成长闭环；
再做祭坛和以太工业，强化基地与自动化；
最后做遗迹、灵能解析和高能级内容。
```

这样既保留了源质升华体系的宏大成长，也保留了神学魔法系统的手感和工程落地，不会变成两套互相抢戏的系统。
