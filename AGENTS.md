# AGENTS.md

1. 优先使用 PowerShell 7。
2. 尽量用加日志的方式解决不确定问题，不要加高频日志。
3. 美术规格参考《异星工厂》的比例规则：地形图块控制世界格尺寸，物品图标控制资产画布和 UI 显示尺寸，不要把所有素材都强行做成同一像素大小。
4. 地形 tile：逻辑格为 `32x32`。可以直接使用 `32x32` tile，也可以使用 `64x64` 高清源并在渲染/导出时按 `0.5` 缩放显示为 1 个 tile。
5. 物品 icon：默认资产规格为 `64x64`。源图可以是高清或较大像素，但必须是单物体、透明背景或可干净抠背景、主体居中、留足透明边距；导出到 `resource/items/**` 时统一为 `64x64` 画布。
6. UI 显示：背包、快捷栏、配方等界面按槽位实际大小缩放显示物品图标，例如显示在 36、40、48 像素槽位内；不要为了 UI 槽位直接把源资产做成 `32x32`。
7. `resource/items/**` 下的游戏内图标应由脚本统一导出，保持一致的画布规格、居中方式、主体最大占比和缩放算法；不要从大图手工随意裁切导致不同图标视觉大小不一致。
8. 如果使用 image 功能生成物品图标，提示词应要求“single centered game item icon, transparent-safe padding, readable at inventory size”。入库前必须通过 `scripts/tools/build_item_icon_assets.py` 或等价脚本统一导出/校验。
